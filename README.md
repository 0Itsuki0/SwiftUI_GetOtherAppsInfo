# SwiftUI_GetOtherAppsInfo
A demo of getting other apps information using iTunes search API, specifically  with either app id or bundle id.

For more details, please refer to my blog [Little Swift UI Tip: Get Info Of Other Apps. By App ID, By Bundle Id.](https://medium.com/@itsuki.enjoy/little-swift-ui-tip-get-info-of-other-apps-by-app-id-by-bundle-id-7c1bd4845d24)
